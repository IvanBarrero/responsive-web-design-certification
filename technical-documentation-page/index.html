Introduction

Python is a programming language which blends two important characteristics:

  • Simplicity: Python is recognized as a programming language that is easy to learn. As a consequence it has been learnt by people not only in computer sicence but also from many other fields with not much programming background.
  • Power: even though Python is easy to learn, it has the power of any formal programming language. It also has a really high performance which can be used in many programming needs.

Another very important characteristic of Python is that its user community is very large and there are many libraries available, ready to be used in many different programming contexts and needs.

Python Interpreter

Python is an interpreted language, so there is no need for compilation and linking. As a consequence, the interpreter can be used interactively, to experiment with code dynamically or to use it as a handy calculator.

Once installed, you can use python's interactive mode by typing: python or python3.8, where 3.8 is the version installed in your machine. After this, the prompt accepts python code to be executed, for instance:

>>> a = 3
>>> b = 5
>>> a+b
8
